# 01 - Requisitos Funcionais

## Empresa Transportadora

* Criar utilizadores.
* Gerir clientes.
* Gerir motoristas.
* Gerir camiões.
* Registar cargas.
* Criar viagens.
* Atribuir motorista.
* Atribuir camião.
* Acompanhar localização.
* Ver mapa.
* Encerrar viagens.
* Consultar relatórios.

## Motorista

* Ver viagens atribuídas.
* Confirmar recolha.
* Iniciar viagem.
* Enviar localização GPS.
* Reportar incidente.
* Confirmar entrega.
* Anexar fotografia da entrega.
* Adicionar observações.

## Cliente

* Ver as suas cargas.
* Consultar estado.
* Acompanhar no mapa.
* Ver histórico.
* Confirmar receção, quando aplicável.

## Módulos funcionais

* Dashboard administrativo.
* Gestão de clientes.
* Gestão de motoristas.
* Gestão de camiões.
* Gestão de cargas.
* Gestão de viagens.
* Rastreamento GPS.
* Incidentes.
* Entregas.
* Notificações.
* Relatórios.
