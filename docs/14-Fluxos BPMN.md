# 14 - Fluxos BPMN

## Fluxo principal da carga

1. Empresa cria a carga.
2. Empresa cria a viagem.
3. Empresa atribui camião.
4. Empresa atribui motorista.
5. Motorista recebe notificação.
6. Motorista confirma recolha.
7. Sistema muda para "Em Trânsito".
8. GPS começa a transmitir localização.
9. Cliente acompanha no mapa.
10. Motorista confirma entrega.
11. Empresa encerra a viagem.
12. Histórico fica disponível.

## Fluxo de incidente

1. Motorista identifica problema durante a viagem.
2. Motorista regista incidente.
3. Sistema guarda localização, data, hora, fotografias e observações.
4. Empresa recebe notificação.
5. Cliente pode ser notificado quando o incidente impactar a entrega.

## Fluxo de entrega

1. Motorista chega ao destino.
2. Motorista confirma entrega.
3. Motorista informa recebedor.
4. Motorista anexa fotografia.
5. Sistema atualiza a carga para entregue.
6. Empresa e cliente recebem notificação.
