# 23 - Estrutura do Repositório

## Diretórios principais

```text
apps/
apps-mobile/
packages/
docs/
scripts/
```

## apps

Aplicações web e serviços principais, como backend e painel administrativo.

## apps-mobile

Aplicações mobile, incluindo o app do motorista.

## packages

Pacotes partilhados entre aplicações, como tipos, validações, configurações e utilitários.

## docs

Documentação funcional, técnica, arquitetura, decisões, testes e manuais.

## scripts

Scripts auxiliares de desenvolvimento, automação e manutenção.
